== Description ==

This plugin makes it easy to install and use SnapEngage Live Chat on your WordPress site. **Please note:** You will need to have an active SnapEngage account to make use of this plugin. 

If you're not already a SnapEngage user, we offer 15-day Free Trials.

== Installation ==

= Automatic Installation: =

* You can install this plugin directly from your WordPress dashboard's 'Plugins' section.

= Manual Installation: =
  
* Unzip contents of this .zip file to your `/wp-content/plugins/` directory.

= After Installation: =

1. Activate the plugin through the 'Plugins' section in WordPress.
2. Go to `Settings >> SnapEngage`
3. Check enable. Type in your [SnapEngage Widget ID](https://www.snapengage.com/getwidgetid). Click save.
4. Log in to the [SnapEngage Web Client](https://www.snapengage.com/client).
5. Get ready to chat with your visitors!

For more info about using SnapEngage, please see the [Getting Started section](http://help.snapengage.com/getting-started/) of our Support Knowledgebase.
